# Induk Panduan Pengguna (User Guide Hub)
## SIM LPPM ITSNU – "The Accountant of Research"

Selamat datang di pusat dokumentasi SIM LPPM ITSNU. Pilih panduan yang sesuai dengan peran Anda untuk mempelajari langkah-langkah penggunaan sistem secara praktis.

---

### [!IMPORTANT]
Sistem ini menggunakan alur kerja terintegrasi. Pastikan Anda memahami peran Anda sesuai dengan Keputusan Rektor/Ketua LPPM.

---

### Daftar Panduan Berdasarkan Peran

| Peran | Fokus Utama | Tautan Panduan |
| :--- | :--- | :--- |
| **Dosen** | Pengajuan proposal & Laporan luaran | [Panduan Dosen](panduan-dosen.md) |
| **Dekan** | Persetujuan awal tingkat fakultas | [Panduan Dekan](panduan-dekan.md) |
| **Reviewer** | Penilaian pakar & Rekomendasi | [Panduan Reviewer](panduan-reviewer.md) |
| **Admin LPPM** | Pengelolaan data & Batas anggaran | [Panduan Admin LPPM](panduan-admin-lppm.md) |
| **Kepala LPPM** | Keputusan final & Monitoring IKU | [Panduan Kepala LPPM](panduan-kepala-lppm.md) |
| **Rektor** | Pengawasan strategis & Analitik data | [Panduan Rektor](panduan-rektor.md) |

---

### Persiapan Sistem (System Requirements)
Untuk pengalaman terbaik, pastikan Anda menggunakan:
- **Browser**: Google Chrome, Mozilla Firefox, atau Microsoft Edge versi terbaru.
- **Koneksi**: Internet stabil untuk pengunggahan berkas proposal (PDF).
- **Resolusi**: Minimal 1280x720 px untuk tampilan dashboard yang optimal.
