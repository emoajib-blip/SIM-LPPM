## Bab 1: Pendahuluan
Admin LPPM bertanggung jawab atas pemeliharaan data sistem, pengaturan anggaran, dan koordinasi alur kerja antara dosen dan reviewer.

---

## Bab 2: Memulai (Getting Started)

### Akses & Login
1.  Gunakan akun Administrator untuk login.
2.  Akses menu **Settings** dan **Master Data** untuk administrasi utama.

## Bab 3: Panduan Fitur

### 3.1 Mengatur Batas Anggaran (Budget Caps)
1.  Buka menu **Settings > Budget Caps**.
2.  Pilih tahun anggaran aktif.
3.  Masukkan nilai maksimal untuk tiap skema (Penelitian Dasar, Terapan, dll).
4.  Simpan untuk mengaktifkan validasi otomatis pada input dosen.

### 3.2 Penugasan Reviewer
1.  Pilih menu **Penugasan Reviewer**.
2.  Cari proposal yang sudah lolos tahap Dekan.
3.  Pilih reviewer yang sesuai dengan bidang ilmu proposal.
4.  Klik **Simpan Penugasan**.

---

## Bab 4: Troubleshooting & FAQ
- **T: Kenapa dosen tidak bisa mengedit proposal?**
  J: Jika status sudah `SUBMITTED`, proposal terkunci. Ubah status kembali ke `DRAFT` jika diperlukan revisi manual.
- **T: Input anggaran dosen tidak lolos validasi?**
  J: Cek apakah nilai Budget Cap untuk skema tersebut sudah diisi di menu Settings.

## Lampiran
### Glosarium
- **Budget Cap**: Pagu anggaran maksimal.
- **Master Data**: Data referensi sistem (Fakultas, Prodi, Bidang Ilmu).
