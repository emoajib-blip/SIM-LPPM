1. TUJUAN MODUL

Modul ini dirancang untuk mengintegrasikan sistem informasi manajemen hibah internal LPPM yang sudah ada dengan target pencapaian Indikator Kinerja Utama (IKU) Perguruan Tinggi berdasarkan Keputusan Menteri Pendidikan Tinggi, Sains, dan Teknologi Nomor 358/M/KEP/2025, dengan perluasan cakupan publikasi pada jurnal nasional terakreditasi SINTA 1–6.

Tujuan Implementasi:

  • Memastikan setiap hibah penelitian dan pengabdian yang didanai berkontribusi langsung pada pencapaian IKU, baik internasional (Scopus/WoS) maupun nasional terakreditasi (SINTA).
  • Memudahkan monitoring dan evaluasi capaian IKU secara real-time.
  • Menyediakan data terintegrasi untuk pelaporan ke LLDIKTI dan Kemdiktisaintek.
  • Mendorong dosen untuk menghasilkan luaran yang relevan dengan IKU, termasuk publikasi di jurnal SINTA 1–6.

4.3 Sistem Skoring Proposal Berbasis Kontribusi IKU

// Fungsi perhitungan skor proposal function hitungSkorIKU($proposal) { $skor = 0; $bobot_iku = [ 'IKU-04' => 25, 'IKU-05' => 25, 'IKU-06' => 25, 'IKU-07' => 15, 'IKU-08' => 10 ]; foreach ($proposal->target_iku as $iku) { $skor += $bobot_iku[$iku]; } // Tambahan skor untuk kolaborasi lintas prodi if ($proposal->jumlah_anggota_lintas_prodi > 0) { $skor += 10; } // Tambahan skor untuk keterlibatan mahasiswa if ($proposal->jumlah_mahasiswa_terlibat > 0) { $skor += 5; } return min($skor, 100); // Maksimal 100 }

7.2 Workflow Validasi Luaran (dengan penanganan SINTA)

// Controller Validasi Luaran (bagian hitungSkorIKU diperbarui) private function hitungSkorIKU($luaran) { $skor = 0; foreach ($luaran->iku as $iku) { // Bobot dasar per IKU $bobot = [ 'IKU-04' => 0.25, 'IKU-05' => 0.25, 'IKU-06' => 0.25, 'IKU-07' => 0.15, 'IKU-08' => 0.10 ]; // Cek sub-indikator berdasarkan jenis luaran if ($luaran->jenis_luaran == 'publikasi' || $luaran->jenis_luaran == 'prosiding') { $detail = json_decode($luaran->detail); $jenis = $detail->jenis_jurnal ?? ''; // Mapping bobot berdasarkan jenis jurnal $bobot_sub = 0; if (strpos($jenis, 'Q1') !== false) $bobot_sub = 1.2; elseif (strpos($jenis, 'Q2') !== false) $bobot_sub = 1.0; elseif (strpos($jenis, 'Q3') !== false) $bobot_sub = 0.75; elseif (strpos($jenis, 'Q4') !== false) $bobot_sub = 0.5; elseif (strpos($jenis, 'Prosiding') !== false) $bobot_sub = 0.25; elseif (strpos($jenis, 'SINTA 1') !== false) $bobot_sub = 0.8; elseif (strpos($jenis, 'SINTA 2') !== false) $bobot_sub = 0.7; elseif (strpos($jenis, 'SINTA 3') !== false) $bobot_sub = 0.6; elseif (strpos($jenis, 'SINTA 4') !== false) $bobot_sub = 0.5; elseif (strpos($jenis, 'SINTA 5') !== false) $bobot_sub = 0.4; elseif (strpos($jenis, 'SINTA 6') !== false) $bobot_sub = 0.3; else $bobot_sub = 0.15; // jurnal nasional lain $skor += $bobot[$iku->kode_iku] * $bobot_sub; } else { // Luaran non-publikasi (HKI, produk, dll) $skor += $bobot[$iku->kode_iku]; } } return min($skor, 1); // Maksimal 1 (100%) }

// Method untuk validasi peringkat SINTA berdasarkan ISSN public function validasiSINTA($issn) { try { $response = Http::withHeaders([ 'Authorization' => 'Bearer ' . $this->apiKey ])->get($this->baseUrl . '/journals', [ 'issn' => $issn ]); if ($response->successful()) { $data = $response->json(); if (!empty($data['journals'])) { // Asumsikan data pertama adalah jurnal yang dimaksud $jurnal = $data['journals'][0]; return [ 'nama' => $jurnal['name'], 'sinta_level' => $jurnal['sinta_level'], // 1-6 'akreditasi' => $jurnal['accreditation'], 'issn' => $jurnal['issn'] ]; } } return null; } catch (\Exception $e) { Log::error('Gagal validasi SINTA: ' . $e->getMessage()); return null; } }

PENUTUP

Modul sistem informasi manajemen hibah internal LPPM yang terintegrasi dengan IKU dan telah diperluas mencakup jurnal nasional terakreditasi SINTA 1–6 ini dirancang untuk:

  • Memudahkan dosen dalam memetakan kegiatan penelitian dan pengabdiannya agar selaras dengan target IKU nasional, baik internasional maupun nasional terakreditasi.
  • Memperkuat peran LPPM sebagai pusat data dan validasi capaian kinerja institusi.
  • Menyediakan data real-time bagi pimpinan untuk pengambilan keputusan strategis.
  • Mengotomatisasi pelaporan ke LLDIKTI dan Kemdiktisaintek melalui integrasi API.

Dengan implementasi modul ini, diharapkan setiap rupiah dana hibah yang dikeluarkan dapat memberikan kontribusi maksimal terhadap pencapaian IKU perguruan tinggi, sekaligus meningkatkan kualitas dan daya saing riset dosen di tingkat nasional dan internasional.
